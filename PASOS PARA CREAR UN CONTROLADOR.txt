1.) digitar los comandos que permiten crear controladores y estructura sql.
comandos en una linea: 
php artisan make:model --migration --controller --resource reserva

*make:model genera el modelo para modificar las cualidades de los atributos de el objeto, puede ser que estos esten
en modo oculto o puede habilitar para que sean llenables o para que sean llenados por la maquina.
objeto: reserva(hora,id,nombre,cancha)


*--migration: comando quegenera el archivo que contiene el esquema sql

*--controller: crea el controlador con el contenido estandar.

*--resource: le ayuda generar el controlador con las operaciones basicas de CRUD (es un complemento de controller).
------------------------------------------------------------
2.)SE CREARON LOS ARCHIVOS DE MIGRACION Y MODELO

app/database/migrations

la migracion se encuentra en la carpeta database con el nombre respectivo y el objeto en plural ejemplo reserva 
2020_06_12_035801_create_reservas_table
(ver tipos de datos y ejemplos en los tipos de datos estan al final https://laravel.com/docs/7.x/migrations)
para modificar este archivo
 
el modelo 
app/reserva.php

3.)SE PASA EL ESQUEMA A LA BASE DE DATOS (internamente es ejecuta create table)
*php artisan migrate
*php migrate:refresh ( se usa si las tablas ya estan creadas 
                        y se quiere refrescar la estructura porque se altero la migracion)
